## 具体任务

### 1. SQL 数据库构建
* **创建表：** 运行 SQL 命令以创建表，包括主键、辅助键和外键。
* **填充数据：** 运行填充每个表的 SQL 命令文件。每个表需要有足够数量的元组（数据行）。
* **数据完整性：** 请记住表之间的关系并据此规划数据。你应该能够成功运行这些命令文件，且没有错误和完整性违规。

### 2. 应用程序开发
你需要为“健康追踪个人健康平台 (Health Track Personal Wellness Platform)”数据库开发一个应用程序系统。
以下是需要开发的菜单。必须实现下述所有应用程序功能（如果你愿意，可以添加更多功能和菜单）。

#### 主菜单 (Main Menu)
* 帐户信息 (Account Info)
* 预约 (Book an Appointment) - 与医疗提供者
* 创建健康挑战 (Create a Wellness Challenge) - 供其他用户使用
* 每月健康摘要 (Monthly Health Summary) - 当月的关键指标和预约
* 搜索记录 (Search Records) - 基于用户健康 ID、提供者、预约类型、日期范围等
* 退出 (Sign Out)

#### 帐户功能 (Account Functions)
* 修改个人详情
* 添加/删除电子邮件地址
* 添加/删除电话号码
* 添加/删除/关联医疗保健提供者

#### 摘要功能 (Summary Functions)
* 用户在指定日期范围内的预约总数。
* 每月特定健康指标（例如：体重、血压）的平均值/最小值/最大值。
* 参与者最多的健康挑战。
* 最活跃的用户（记录健康记录最多或完成挑战最多的用户）。
